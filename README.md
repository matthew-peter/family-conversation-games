A collection of conversational games optimized for family gatherings, road trips, and game nights. Zero setup required, works offline (mostly), and designed for mobile.

## Games Included

- **Taboo**: Describe words without saying forbidden ones. Includes categories for Sports, Nature, Science, History, Bible, and Family Life.
- **Charades**: Act out prompts from categories like Animals, Sports, Occupations, and more.
- **Pictionary**: Draw words from categories like Animals, Objects, Food, and more.
- **Ice Breakers**: Fun questions to get conversations started.
- **Riddles & Lateral Thinking**: Brain teasers and puzzles.
- **Word Games**: Contact, Word Chain, Neanderthal.
- **Parlor Games**: Fortunately/Unfortunately, Only Questions, and more.

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **Icons**: React Icons
- **Deployment**: Ready for Vercel

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser.

## Deployment

Push to GitHub and import the project into [Vercel](https://vercel.com).
